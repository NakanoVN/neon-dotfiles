### 2. Disable persistent system logs

### 3. Use zram instead of swap

Install:

```bash
sudo pacman -S zram-generator
```

Then create `/etc/systemd/zram-generator.conf`:

```ini
[zram0]
zram-size = ram / 2
compression-algorithm = zstd
```

Enable it:

```bash
sudo systemctl enable --now /dev/zram0
```

This uses compressed RAM instead of a swap file, avoiding USB writes.
